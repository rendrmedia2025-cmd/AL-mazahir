## Getting Started

### Accessing the Admin Panel

1. Navigate to `https://your-domain.com/admin/login`
2. Enter your admin credentials
3. Click "Sign In"
4. You will be redirected to the admin dashboard

### First Time Setup

After your first login:
1. Update your profile information
2. Review current availability settings
3. Check recent leads
4. Familiarize yourself with the dashboard layout

## Content Management

### Updating Contact Information

1. Navigate to **Content Management**
2. Select **Contact Settings**
3. Update phone numbers, WhatsApp details, or email addresses
4. Click **Save Changes**

### Managing Hero Section

1. Go to **Content Management** > **Hero Section**
2. Edit the main headline text
3. Update call-to-action button labels
4. Preview changes before saving
5. Click **Update Hero Section**

### Category Management

Enable or disable product categories:
1. Go to **Content Management** > **Categories**
2. Toggle categories on/off using the switches
3. Reorder categories by dragging them
4. Changes are saved automatically

## Lead Management

### Viewing Leads

1. Navigate to **Lead Management**
2. View all incoming inquiries in the main table
3. Use filters to find specific leads:
   - Filter by date range
   - Filter by product category
   - Filter by urgency level
   - Filter by status

### Lead Details

Click on any lead to view:
- Complete contact information
- Product category of interest
- Urgency level and quantity estimate
- Source section and device information
- Full message content
- Lead intelligence metadata

### Exporting Leads

1. Select leads using checkboxes
2. Click **Export Selected** or **Export All**
3. Choose format (CSV or Excel)
4. Download will start automatically

### Lead Status Management

Update lead status to track progress:
1. Open lead details
2. Change status from dropdown:
   - New
   - Contacted
   - Qualified
   - Converted
   - Closed
3. Add notes about follow-up actions
4. Save changes

## Troubleshooting

### Common Issues

#### Cannot Login
**Problem**: Login fails with correct credentials
**Solution**:
1. Clear browser cache and cookies
2. Try incognito/private browsing mode
3. Check if account is active
4. Contact system administrator

#### Changes Not Appearing
**Problem**: Updates not visible on website
**Solution**:
1. Wait 5 minutes for cache refresh
2. Check if changes were saved successfully
3. Clear browser cache
4. Verify feature flags are enabled

#### Slow Performance
**Problem**: Admin panel loading slowly
**Solution**:
1. Check internet connection
2. Close unnecessary browser tabs
3. Clear browser cache
4. Check system status page

#### Export Not Working
**Problem**: Lead export fails or downloads empty file
**Solution**:
1. Ensure leads are selected
2. Check browser popup blockers
3. Try different export format
4. Refresh page and retry
